# 발더스 게이트

## 한 줄 요약

발더스 게이트은 공개 로그에서 반복적으로 언급되는 지역입니다.

## 기본 정보

| 항목 | 내용 |
|---|---|
| 분류 | 지역 |
| 첫 등장 | session-001 |
| 현재 상태 | 확인 필요 |
| 확신도 | medium |

## 상세 정보

- 공개 로그에서 반복적으로 확인되는 항목입니다.
- 현재 문서는 자동 초벌이며, 세부 설정은 출처 로그 대조 후 보강해야 합니다.
