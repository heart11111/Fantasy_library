# 도시

## 한 줄 요약

도시은 공개 로그에서 반복적으로 언급되는 지역입니다.

## 기본 정보

| 항목 | 내용 |
|---|---|
| 분류 | 지역 |
| 첫 등장 | session-001 |
| 현재 상태 | 확인 필요 |
| 확신도 | medium |

## 상세 정보

- 공개 로그에서 반복적으로 확인되는 항목입니다.
- 현재 문서는 자동 초벌이며, 세부 설정은 출처 로그 대조 후 보강해야 합니다.

## 관계

| 대상 | 관계 | 설명 |
|---|---|---|
| 확인 필요 | 확인 필요 | 명확한 관계는 2차 정리에서 보강합니다. |

## 주요 행적

| 시점 | 행적 |
|---|---|
| session-001 | 경쟁자배제 로그에서 언급됨 |
| session-002 | 궁지에 몰린 쥐 로그에서 언급됨 |
| session-003 | 기로 로그에서 언급됨 |
| session-005 | 더 월드 - 24 작은 마법사의 꿈 로그에서 언급됨 |
| session-006 | 더 월드 29 룩스테라-해골 틈에서 춤을 추다 로그에서 언급됨 |
| session-009 | 더 월드 47 룩스테라-연구자료 호송대를 찾아서 로그에서 언급됨 |
| session-010 | 더월드 25 - 룩스테라-1 도적이 숨은 숲 로그에서 언급됨 |
| session-011 | 더월드 30 룩스테라-해골의 뒤에 보이는 것은 로그에서 언급됨 |
| session-012 | 더월드 33 룩스테라-붉은 닻의 심지 로그에서 언급됨 |

| session-015 | 드워프 유적 로그에서 언급됨 |

## 확인 필요

- 정식 소속, 현재 위치, 장기 목표는 로그 대조가 필요합니다.
- 농담, 메타 발언, 플레이어 추측은 정사로 확정하지 않았습니다.
